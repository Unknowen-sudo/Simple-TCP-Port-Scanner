# ⚡ Async-Scan: Simple TCP Port Scanner

An entry-level network auditing tool built in Python using the native `socket` library. This project marks **Project 01** of the VectorCore Architecture roadmap.

## 🚀 Features
* Interactively requests target IP address.
* Scans a customizable range of TCP ports (Default: 1-100).
* Implements robust timeout handles to optimize speed on closed ports.
* Safe and resource-efficient connection handling.

## 🛠️ Requirements
* Python 3.x

## 💻 How To Run on windows 
1. Cloning the repository And Runing It:
```bash
git clone https://github.com/Unknowen-sudo/Simple-TCP-Port-Scanner.git
cd Simple-TCP-Port-Scanner
python Main.py
```
